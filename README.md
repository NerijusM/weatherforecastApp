# weatherforecastApp
Orų prognozė visuose Lietuvos miestuose

Užduotis:

„Sukurti Web arba Win (WPF) aplikaciją (pasirinktinai). Aplikacija turi grafiškai atvaizduoti bet kokio miesto orų prognozės duomenis. Duomenys turi būti paimami iš pasirinkto Web API. Pateikti aplikacijos Source Code. Užduotyje būtina panaudoti .NET. Duomenų paėmimas iš API turi būti realizuojamas BackEnd'e.“

Sprendimas:

Panaudotas MeteoLt Api adresu: https://api.meteo.lt

Sukurti du projektai:
1. ForecastCore
2. ForecastWebApp

Palaidžiamasis projektas ForecastWebApp.
